# BIN2DEC


wavファイルのバイナリテキストとして出力されたファイルから1周期分のバイト列を10進数のバイト列としてoutput.txtに出力します．

バイナリテキストのファイルをcolab上にアップロードしてください．

**また，バイナリテキストのファイルは1つだけにしてください.**

具体的には．dataを表すバイト列の行の次から1周期分を取得して，符号付16進数から10進数へ変換しています.
